📐 Неравенства с одной переменной

8 класс · Системы неравенств · Обозначения · Решение

📖 Термины и обозначения

Неравенство — выражение вида a > b, a < b, a ≥ b, a ≤ b.
Решение неравенства — значение переменной, при котором неравенство верно.
Система неравенств — несколько неравенств, которые должны выполняться одновременно.
Числовой промежуток — множество чисел, удовлетворяющих неравенству.
Интервал (a; b) — a < x < b (круглые скобки — концы НЕ включены).
Отрезок [a; b] — a ≤ x ≤ b (квадратные скобки — концы включены).
Луч [a; +∞) — x ≥ a; (−∞; a) — x < a.
Пересечение решений — общая часть всех решений системы.

📌 Примеры записи ответов:

x > 5 → (5; +∞)
x ≤ -2 → (-∞; -2]
-3 < x ≤ 4 → (-3; 4]
нет решений → ∅

🧩 Алгоритм решения системы неравенств:

1. Решить каждое неравенство отдельно.
2. Отметить решения на координатной прямой.
3. Найти пересечение (общую часть).
4. Записать ответ в виде промежутка или луча.
